### 上手指南


###### 开发前的配置要求

1. Python 3.10
2. Poetry(非必须)

###### **安装步骤**

1. Clone
   ```sh
   git clone https://github.com/Virace/valorant-audio.git
   ```
2. 安装依赖
   ```sh
   pip install -r requirements.txt
   ```
   or
   ```sh
   poetry install
   ```
